Base types
==========

=====
Score
=====

+--------------------+-------------------------------------------------------+
| Data               | *Range* (optional: 1 or more *Name*; 1 *Score*; 1     |
|                    | *Strand*, 1+ *Metadata*, 1+ *Set*)                    |
+--------------------+-------------------------------------------------------+

*Score* adds an integer or decimal numerical attribute to *Element*, *Interaction* or *Set* compound types.
