Range operations¶

Adjust¶

Input :
1 or 2 integers for “left” and “right” ranges; Element Set
Output :
Element Set
CLI example :
bedops --range L:R --everything ElemSetA > ElemSetB

This operation shrinks or grows Elements by specified integer values.

Positive values expand elements; negative values shrink elements. Any Element with a processed Range with a length 0 or less is discarded.

If the values of L and R are equal and positive, the element grows symmetrically. If the values of L and R are equal and negative, the element shrinks symmetrically.
