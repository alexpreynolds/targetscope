Interaction¶

Range¶

Input
1 or 2 integers for promoter or DHS peak component range(s); Interaction Set; index of desired interaction component (0/promoter or 1/peak)
Output
Interaction Set

This operation shrinks or grows either the promoter or DHS component of specified Interactions by specified integer values.

Positive values for the range component expand elements; negative values shrink elements. A positive left value expands the element on the left edge of its range; conversely, a negative left value shrinks the element on its left edge. The same applies for the right value, to the right edge of the element’s range.

Any Element with a post-processed Range with a length 0 or less causes the entire Interaction to be discarded.

If the values of L and R are equal and positive, the promoter or peak component grows symmetrically. If the values of L and R are equal and negative, the promoter or peak component shrinks symmetrically.

The command-line equivalent would likely be a BEDOPS bedops --range operation on the specified component.

Union¶

Input
1 or more Interaction Sets
Output
Interaction Sets

This operation takes the union of Interactions in a Set, yielding one Set made up of each of the input Interactions. Each Interaction in the output set preserves its attributes. The union of one Interaction Set is the Set itself.
